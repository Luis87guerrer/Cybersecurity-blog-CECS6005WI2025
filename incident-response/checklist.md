# IRP Checklist

## Preparation
- Assign roles (Incident Manager, technical lead, communications)
- Keep offline contact list + key procedures
- Run tabletop exercises

## During the incident
- Detect and confirm scope/impact
- Contain (short-term isolation + long-term controls)
- Preserve evidence and document actions

## After the incident
- Recover systems/services safely
- Post-incident review (lessons learned)
- Update the plan + improve controls
